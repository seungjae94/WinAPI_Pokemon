# 포켓몬 파이어레드 모작

### 게임 소개
- 개요: <a href="https://namu.wiki/w/%ED%8F%AC%EC%BC%93%EB%AA%AC%EC%8A%A4%ED%84%B0%20%ED%8C%8C%EC%9D%B4%EC%96%B4%EB%A0%88%EB%93%9C%C2%B7%EB%A6%AC%ED%94%84%EA%B7%B8%EB%A6%B0">포켓몬스터 파이어레드 버전</a>을 본따서 만들어 본 게임
- 플랫폼: Windows
- 개발 인원: 1인
- 게임 엔진: 자체 엔진 (C++ / Windows API)
- IDE: Visual Studio 2022

### 조작법
- 이동: <kbd>&larr;</kbd><kbd>&uarr;</kbd><kbd>&rarr;</kbd><kbd>&darr;</kbd>
- 확인: <kbd>Z</kbd>
- 취소: <kbd>X</kbd>
- 메뉴창: <kbd>Enter</kbd>

### 치트키
- 2배속 진행: <kbd>Space</kbd>
- 치트 HUD 온오프: <kbd>F2</kbd>
- 데미지 배율 변경: <kbd>Q</kbd>
- 경험치 배율 변경 : <kbd>W</kbd>
- 야생 포켓몬 등장 확률 변경: <kbd>E</kbd>
- 포획률 변경: <kbd>R</kbd>
- 맵 충돌 시각화 기능 온오프: <kbd>O</kbd>
- 맵 이동 (태초마을): <kbd>1</kbd>
- 맵 이동 (상록시티): <kbd>2</kbd>
- 맵 이동 (상록숲): <kbd>3</kbd>
- 맵 이동 (회색시티): <kbd>4</kbd>

### 주의사항
스타팅 포켓몬을 받기 전에 치트를 사용해 다른 마을로 이동한 뒤 수풀에 들어가면 버그가 발생할 수 있습니다.
